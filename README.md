# 🍀 Clover Delivery

Веб-додаток для замовлення їжі — тестове завдання ElifTech School (Advanced level).

## Стек

**Frontend:** React 18, TypeScript, SASS, Vite, Zustand, react-i18next  
**Backend:** Node.js, Express, TypeScript, Mongoose  
**База даних:** MongoDB

## Що реалізовано

**Base:** сторінка магазинів, кошик з валідацією, збереження замовлень у MongoDB.  
**Middle:** адаптивний дизайн, фільтрація товарів за категорією та магазинів за рейтингом, сортування за ціною та назвою.  
**Advanced:** серверна пагінація (6 товарів/сторінка), історія замовлень із пошуком за email або ID, повторне замовлення одним кліком.  
**Додатково:** темна/світла тема, українська/англійська мова, промокоди, toast-сповіщення, skeleton-loader.
